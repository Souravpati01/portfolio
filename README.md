# Personal Portfolio Website

A responsive personal portfolio website created using HTML and CSS.

This project was developed as part of a CSS assignment to demonstrate concepts like Flexbox, Box Model, Responsive Design, Hover Effects, Positioning, and Media Queries.

## Project Features

- Responsive Navigation Bar
- Hero Section with Introduction
- About Me Section
- Skills Section
- Projects Section with 3 Project Cards
- Contact Section
- Responsive Layout for Mobile Devices
- Hover Effects on Buttons and Cards
- Flexbox Layout
- Sticky Navigation Bar

## Concepts Demonstrated

### CSS Selectors
Used class selectors, element selectors, and hover selectors.

### Box Model
Implemented using:
- Margin
- Padding
- Borders

### Flexbox
Used for:

- Navigation bar  
- Skills section  
- Project cards  
- Hero stats section  

### Positioning
Used sticky navigation bar.

### Hover Effects
Added hover animations on:

- Buttons  
- Project Cards  
- Navigation Links

Responsive layout for tablets and mobile devices.
